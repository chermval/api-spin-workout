# API SPIN WORKOUT
NestJS api rest for indoor bike workout

## Install

### Requisites
- Node

`npm i`

## Run the app

### Local/Dev
`npm run start`

#### Watch mode
`npm run start:dev`

### Production
`npm run start:prod`

## Run the test
`npm run test`

### e2e tests
`npm run test:e2e`

### test coverage
`npm run test:cov`

## Usage

| ACTION | METHOD  | URI               | EXAMPLE               | 
|--------|---------|-------------------|-----------------------|
| Validate health of api | `GET`   | `/api/v1/healthcheck` |  http://localhost:3000/api/v1/healthcheck |
